# faf

**Thin alias for `faf-cli`.** Same CLI, shorter name.

```bash
bunx faf init
bunx faf auto
bunx faf go
bunx faf wjttc
```

This package resolves [`faf-cli`](https://npmjs.com/package/faf-cli) at install time and forwards args through. All docs, all commands, all features live in the canonical CLI.

- **Canonical CLI:** https://npmjs.com/package/faf-cli
- **Spec:** https://faf.one
- **Issues:** https://github.com/Wolfe-Jam/faf-cli/issues
- **Format:** IANA-registered `application/vnd.faf+yaml`

> FAF defines. MD instructs. AI codes.
