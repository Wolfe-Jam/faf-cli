How to Install Claude Code Skills

Get 32 championship-grade AI workflow skills in your Claude Code in under 60 seconds. These skills include /pubpro for publishing, /downloads for analytics, and complete workflow automation.

Step 1: Install MCP Server

npm install -g claude-faf-mcp

Provides 33 tools for persistent AI context and project understanding.

Step 2: Install Skills Collection

faf setup --complete

Adds 32 championship-grade workflow skills to your Claude Code.

Verify Installation

Open Claude Code and type / - you should see skills like:

  • /pubpro - Zero-mistakes publishing protocol
  • /downloads - Multi-registry analytics
  • /faf-expert - Deep .faf knowledge expert
  • /commit - FAF-powered git commits

Individual Collections

Install specific skill collections:

faf skills install --bundle=core faf skills install --bundle=development faf skills install --bundle=publishing

Troubleshooting

Skills not appearing?

  1. Restart Claude Code completely
  2. Wait 10-20 seconds for skills to load
  3. Type / and scroll down past built-in commands
